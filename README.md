# Python酒店管理系统 python973

## 项目概述

本项目是基于Python语言的酒店管理系统，采用了Django框架和Vue.js进行前后端开发，使用MySQL作为数据库存储。系统角色包括管理员和普通用户，实现了完善的权限管理。

## 技术栈

- **后端**: Python + Django
- **前端**: Vue.js
- **数据库**: MySQL

## 系统角色

- 管理员：负责系统维护、用户管理和酒店业务处理等。
- 用户：可以进行房间预订、查询订单等。

## 功能模块

系统主要功能模块包括：

- 用户注册与登录
- 房间预订与查询
- 订单管理
- 用户管理
- 系统设置

## 运行环境

- 后端：Python 3+，Django 2.2+
- 前端：Node.js (版本高于14)，Vue.js
- 数据库：MySQL 5.7+

## 部署步骤

1. 环境准备：安装Python、MySQL、Node.js等开发环境。
2. 数据库配置：根据提供的SQL脚本初始化数据库。
3. 后端部署：配置Django项目，并运行。
4. 前端部署：编译Vue项目，并运行。
